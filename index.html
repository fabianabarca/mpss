    Estas son notas de clase del curso, acompañados de ejercicios de la teoría y de programación.

    Asignaciones

    1. (2%) ¡Repartir estrellas! Visitar GitHub con su cuenta (la misma que será usada en el proyecto) y colocar una estrella en los siguientes repositorios relacionados con el curso:

      mpss

      Este repositorio de notas de clase

      pyx

      Tutoriales introductorios de Python y sus herramientas para análisis de datos

      kalouk

      Presentaciones web para matemáticas y programación, desarrollado para este curso

      slidev-theme-kalouk

      Tema visual de Kalouk para Slidev, la plataforma de presentaciones web base

      slidev-addon-kalouk

      Funcionalidades de Kalouk para Slidev, la plataforma de presentaciones web base

    2. (8%) Realizar una transcripción de las presentaciones según la distribución de personas y documentos de la tabla siguiente.

    ¿Qué debo transcribir y cómo?

    En cada documento dentro de docs/ hay una indicación de las secciones específicas para transcribir. Pueden ver una explicación varios ejemplos en las notas sobre la transcripción.

    Para saber cómo ejecutar localmente el repositorio, cómo ver en su navegador esta documentación y cómo editar en un ambiente integrado de desarrollo (VS Code, por ejemplo), pueden consultar el documento HOWTO con las instrucciones básicas.
